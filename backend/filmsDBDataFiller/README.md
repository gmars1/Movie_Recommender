# Movie_Rec (1.1) - IMDb Top 250 Parser

Этот проект собирает данные о топ-250 фильмах с IMDb, включая русское название, рейтинг, жанр и актеров, и анализирует частоту появления актеров.

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше.

2. Откройте терминал или командную строку и перейдите в папку проекта:

cd "путь/до/Movie_Rec (1.1)"

> Обратите внимание на кавычки, если в пути есть пробелы или скобки.

3. Установите зависимости:

pip install -r requirements.txt

## Запуск

1. Соберите данные с IMDb:

python main_import.py

Данные сохранятся в файл `top_250.xlsx` в этой же папке.

2. Проанализируйте частоту появления актеров:

python count_actors.py

Результат будет сохранён в `count_actors.xlsx`.

## Зависимости

- aiohttp
- beautifulsoup4
- pandas
